# AI Interview Simulator

An AI-based web application that generates personalized interview questions based on user skills. It helps users practice and improve interview performance.

---

## Features
- Generate interview questions from user skills  
- Simple and interactive UI  
- Future scope: AI feedback, resume parsing, voice interaction
